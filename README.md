# Portfolio Website

A simple portfolio website built with React, Vite, Tailwind CSS, and React Router DOM.

## Features

- Home Page
- About Page
- Contact Page
- Responsive Design
- React Router Navigation
- Contact Form
- Local Storage

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- JavaScript
